Tecnologias Backend

¿Que son?

Son la parte de desarrollo web, se encarga de la logica para que la misma funcione. En si son las acciones que pasan en la web, pero que logramos ver, un ejemplo podria ser la comunicación misma del servidor con la pagina

Ciertas funciones:

  • El Desarrollo de funciones que simplifiquen el proceso de desarrollo.
  • Acciones de lógica.
  • Conexión con bases de datos.
  • Permite crear aplicaciones multiplataforma.
  • Uso de librerías del servidor web (por ejemplo para implementar temas de caché o para comprimir las imágenes de la web).

Tipos de tecnologias para backend

Podemos mencionar algunos de los lenguajes en los cuales se podria realizar esto, para ello se indica los siguientes 4 lenguajes:

  1. Python
  2. PHP
  3. Golang
  4. Ruby

PYTHON

Python es un lenguaje de programación muy diverso, con esto me refiero a que se lo que usar para varios tipos o propósitos como, aplicaciones web, ciencias de datos y hasta Machine Learning, este mismo lenguaje el cual tiene una popularidad en incremento es uno de los preferidos por los usuarios, debido a que es más fácil de aprender, mas esto no le quita el ser uno de los lenguajes de más alto nivel, ya que es muy utilizado para la inteligencia artificial. (Marzal, Gracia, & Garcia, 2014)

Caracteristicas Principales:

  • Lenguaje fácil de utilizar, su sintaxis es comprensible y generalmente su código es más corto.
  • Al ser un lenguaje de alto nivel, lo hace comprensible, ya que se asemeja mas al idioma de las personas, esto provoca una disminución en administración de la memoria.
  • Para las variables, no se necesitan ser anunciadas, pueden ser llamadas e interpretadas directamente, esto aumenta la rapidez de su programación
  • Interpreta el código línea por línea, lo cual ayuda a resolver el error con mas rapidez, ya que se detiene justo en la línea donde se encuentra dicho error.
  • Permite la programación orientada a objetos

PHP

PHP es un lenguaje de programación altamente utilizado y con una trayectoria muy extensa, aunque poco a poco ha ido en desuso o está siendo reemplazada, no deja de estar muy vigente en la actualidad, haciendo que el lenguaje siga perdurando con el tiempo, este lenguaje es solo utilizado para paginas web, su desarrollo no es versátil para otros usos que no sean web y es de cliente – servidor. (Dimes, 2016)

Algo que ahí que tener en claro es que PHP a trabajar con Framework, lo hace más fácil de escribir, ya que por sí solo sería PHP puro.

Caracteristicas Principales:

  • Al ser de código abierto nos indica que todos los módulos están disponibles y se pueden personalizar libremente.
  • Debido a que fue el primero en ser cliente-servidor, la comunidad es inmensa, así que cualquier duda se podría solventar en foros.
  • Soporta programación orientada a objetos.
  • Unos de los servicios para alojamiento tenemos el ASP.NET, que, aunque PHP es libre, el uso del servicio es muy aparte y puede llegar a tener costo.
  • Su documentación es inmensa.
Tabla Informativa
Ventajas: Desventajas:
  • Es muy extensa en su nomenclatura y desarrollo
  • Es de aprendizaje común o básico, lo cual lo hace fácil de aprender.
  • Lenguaje de plataforma múltiple. (ONOFRE AVILA, 2021)
  • Se necesita de un servidor web para que funcione
  • Solo es útil para desarrollo web
  • Su configuración no puede ser la más amigable, con ello podríamos dejar abiertas ciertos tipos de seguridad de este.
Tipo de Licencia: Curva de aprendizaje:
Open Source Su curva de aprendizaje es baja, a pesar de tener una nomenclatura mas extensa, no es difícil aprender el mismo, puedes pasar de lo básico a lo mas complejo en poco tiempo, aparte de esto podría a ayudar al usuario a desarrollar una mejor lógica en programación.

GOLANG

Mas conocido como GO, es un lenguaje de programación inspirado en C, con la diferencia que tiene seguridad de memoria y la recolección de basura, utilizado para apps, compilación de plataforma cruzada, paginas web, inteligencia artificial, entre otros más usos que nos brinda este lenguaje, pero uno de sus auges en sí, es la solución que brinda cuando el uso de servicios en la nube se refiere.

Caracteristicas Principales:

  • Como ya había mencionado, una de sus características es la recolección de basura.
  • Es concurrente
  • Tiene la característica del multiprocesador
  • Al igual que muchas otras, tiene o conlleva a una programación orientada a objetos.
  • Es similar a la sintaxis de C, al ser concisa y claro.
Tabla Informativa
Ventajas: Desventajas:
  • Al ser un lenguaje robusto, no lo hace un código complicado, ya que fue diseñado para ser lo más amigable al programador, haciendo de programación un poco más simple
  • Conlleva una gran librería, lo que nos da paso la BIGDATA.
  • Tiene un buen rendimiento, y permite múltiples conexiones.(Muradas, 2020)
  • Al ser un lenguaje no tan conocido o de poco uso, no cuenta con gran documentación o manuales en sí.
  • No permite hacer el uso de herencia directamente, para ello se necesita primero usar otra función.
  • Tiene carencia de programadores al ser un lenguaje con poco recorrido.
Tipo de Licencia: Curva de aprendizaje:
Open Source Su curva de aprendizaje es plana, lo cual no lo hace tampoco tan difícil de aprender.

RUBY

Es un lenguaje de programación inspirado en varios lenguajes como Perl, Smalltalk, Eiffel, Ada, Lisp, es un lenguaje multiplataforma que gracias a un Framework, puede facilitar el desarrollo de aplicaciones online para redes sociales y más, también orientado a objetos, al ser un lenguaje con años de ser liberado, cuenta con una extensa comunidad, la cual en los años a liberado gran variedad de documentación y soluciones en los foros, es un programa que busca la flexibilidad y al igual que otros, que su lenguaje sea lo mas humano posible en pocas palabras, o de entendimiento simple. (ruby, 2003)

Caracteristicas Principales:

  • Facilita el manejo de posibles errores, con el manejo de excepciones.
  • Si el sistema lo permite, puede cargar bibliotecas dinámicas.
  • Es portable, aunque se usa mas habitualmente en sistemas Linux.
  • Soporta múltiples bases de datos.
  • Su sintaxis es simple.

Ventajas: Desventajas:
  • Su código es totalmente libre, por o tanto, no necesitas ningún pago para su uso
  • Programación orientada a objetos, que facilita la misma para desarrollos en la web.
  • Puede ser modificado a gusto del usuario.
  • No tiene muchas Framework
  • Sus actualizaciones cada vez son más escasas.
  • Para programación web, puede llegar a ser un poco tedioso el acostumbrarse a la misma.
